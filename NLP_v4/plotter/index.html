Sentiment Analysis
Sentiment Analysis refers to the use of concepts of natural language processing to extracts subjective information from sourse material. It refers to mark polarity to the sentence based on the contents and relation among the words.
Sentiment Analysis On Camera Reviews
The domain of the sentiment analysis is huge so in order to learn, study and develop the current state of the sentiment analysis we picked the most common genre in sentiment analysis that of the reviews and in specific camera review. We try to give the sentiment corresponding to some given feature of the camera product based on two common appraoches in NLP
  1. Rule Based Approach
  2. Machine learning based classification
What We Did
We designed two diffent types of sentiment analyser. One uses rule based appraoch and deals with the structure of sentence and the properties of the sentence, Second one based on machine learning that uses naive bayes classifier to classify the aspect and the sentiment of a given sentence.
